Lumberjacks vs Beavers

An HTML5 Real-Time Strategy Game

OSU CS 467 Summer 2017 Team Grus Capstone Project

By Joshua Couture, Steven Lim, and Benjamin Olson

New Game - Easy Mode
New Game - Hard Mode
Load Game

  • How To Play

  • Gather resources from trees and berry bushes to build new units.
  • Select structures to build new units.
  • Build new structures to gather resources and build new units.
  • Move your units to enemy units and structures to attack them.
  • Move your units to a friendly structure to slowly heal them.
  • Destroy all enemy structures to win the game.

  • Controls

  • Arrow Keys - Move Camera
  • Left Click - Select a unit or structure.
  • Right Click - Move unit.
  • Ctrl + Left Click - Use twice to select all unites between the two clicks
  • Ctrl + Right Click - When a unit is selected Ctrl + Right Clicking a bush or tree will assign that unit to gather resources
